# Caradelibro - Launcher para Adquisición de Datos SOCMINT en Facebook


Este script está diseñado para realizar análisis SOCMINT en Facebook, facilitando la adquisición de datos y la interacción con cuentas a través de un navegador web.

---

## Funciones del Script:

Este script además permite realizar las siguientes acciones:

- **Ver amigos de un perfil**: Muestra la lista de amistades, incluyendo parientes y conocidos.
- **Ver likes de un perfil**: Extrae la información sobre gustos y hobbies del perfil.
- **Ver amigos en común**: Te permite identificar amigos en común entre dos perfiles.

Cada una de estas opciones está diseñada para ofrecer datos clave y agilizar el análisis SOCMINT.
